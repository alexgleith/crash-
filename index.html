Explore car accidents across Tasmania over the last ten years. Try out the three different ways of visualising crashes below

These three maps have been created mostly using off the shelf tools using CartoDB, which is a cloud base web-mapping system. The accident information was sourced from the Tasmanian Department of State Growth's open data repository on data.gov.au.

I hope you enjoy the maps and that they might provide some insight into this large dataset.
